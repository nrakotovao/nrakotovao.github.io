Composer

Sur le Mac, PHP se trouve sur /Applications/XAMPP/xamppfiles/bin/php-8.2.4. Donc au lieu de php, il faut utiliser ce path complet.
php composer.phar show -i Affiche la liste des packages installés.
php composer.phar require phpoffice/phpword Installe le package PHPWord, par exemple.
php composer.phar update phpunit/phpunit Met à jour individuellement un package.
php composer.phar update phpunit/phpunit --with-all-dependencies Met à jour individuellement un package, et met à jour les dépendances.
