    Fampiavahana

    Misy fikirana vitsy ireto mba hampiavaka ny éditeur an'i Godot sy hanamora ny fampiasana azy.

    # Manova ny THEME : lokon'ny éditeur
    Mankao @ menoa Editor > Editor settings...
    Alefa ny option "Advanced settings" (ery ankavanana).
    Safidiana ny loko tiana oa @ General > Interface > Theme > Preset.
    Mety mila miandry kely mandra-panovan'i Godot ny theme rehefa avy misafidy.

    # Mi-scroll any ambanin'ny soratra farany @ kaody
    Mankao @ menoa Editor > Editor settings... > General > Text editor > Behavior
    Alefa ny option "Scroll past end of file".

    Manapaka soratra lava (word wrap)
    Mankao @ menoa Editor > Editor settings... > General > Text editor > Appearance
    Soloina "Boundary" ny valeran'ny option "Word wrap".
