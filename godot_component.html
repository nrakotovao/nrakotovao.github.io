    ScrollContainer

    Ny ScrollContainer dia composant manome fonctionnalité scroll ho an'ny composant ao anatiny. Izany hoe atao ao anatiny izay composant mety ho lehibe noho ny alehibe azony tànana. Raha vao mihoatra ny ilay halehibe dia mandeha ny scroll ary miseho ireo barres de scroll. 

    # GDScript:
var scroll_container = ScrollContainer.new()
scroll_container.horizontal_scroll_mode = ScrollContainer.SCROLL_MODE_AUTO   #scroll horizontal alefa
scroll_container.get_h_scroll_bar().modulate = Color(0, 0, 0, 0)     #tsy aseho ny indicateur scroll na alefa aza ny scroll
scroll_container.vertical_scroll_mode = ScrollContainer.SCROLL_MODE_DISABLED    #scroll vertical tsy alefa
scroll_container.mouse_filter = Control.MOUSE_FILTER_PASS    #avela miampita @ component parent ny mouse event
panel_gris.add_child(scroll_container)    #ampidirina @ component tree
