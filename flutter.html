    Généralité

    Organization du code : bien séparer les codes Dart dans /src, puis /src/widgets, /src/data et /src/models...

    Widgets standards:

    • Card - Conteneur avec bord pour un composant
    • Column - Conteneur de composants agencés en vertical
    • Container - Contenant pour un seul composant (child)
    • ListView - Vue scrollable horizontallement (par défaut) ou verticalement
    • Row - Conteneur de composants agencés en horizontal
    • Scaffold - Squellette d'une application
    • Text - Composant texte

    Row

    Conteneur horizontal

    Row(
  children: <Widget>[...],
);

    Listview

    Modifier le sens de scroll (par défaut verticalement)

    ListView(
  children: <Widget>[...],
  scrollDirection: Axis.horizontal,
);

    Autres propriétés intéressantes

    scrollDirection: Axis.horizontal,

    Orientation

    Limiter l'orientation de l'application à portrait seulement (bloquer le mode paysage) :

    SystemChrome.setPreferredOrientations([DeviceOrientation.portraitUp, DeviceOrientation.portraitDown]);
