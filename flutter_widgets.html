    Conteneurs

    MaterialApp
    Conteneur principal pour material design. Il est sensé contenir un scaffold comme "home".
    home: (scaffold ou widget), theme: ThemeData(), 

Scaffold
    Conteneur secondaire contenant les éléments standards d'un app suivant le material design.
    appBar: AppBar(), body: (widget), backgroundColor: Colors.white, floatingActionButton: (bouton flottant), bottomNavigationBar: (barre navigation bas),  

Container
    Conteneur générale.
    children: [], width: 100, height: 100, color: Colors.blue,  

Row
    Place ses widgets enfants horizontalement.
    children: [], mainAxisAlignment: start/end/center/spaceBetween/spaceAround/spaceEvenly, crossAxisAlignment: start/center/end/stretch/baseline, mainAxisSize: MainAxisSize.max/min, 

Column
    Place ses widgets enfants verticalement.
    children: [], mainAxisAlignment: start/end/center/spaceBetween/spaceAround/spaceEvenly, crossAxisAlignment: start/center/end/stretch/baseline, mainAxisSize: MainAxisSize.max/min, 

Expanded
    Enfant direct d'un Row ou Column, occupe l'espace disponible du MainAxis.
    child: {widget}, flex: 2 {facteur d'occupation si plusieurs expanded}
     

    Scrolls

    ListView
    Le plus courant. Widget scrollable affichant ses enfants l'une après l'autre dans la direction de scroll.
    scrollDirection: Axis.vertical/horizontal, padding: (espace autour contenus), 
	Utiliser ListView.builder() pour les listes longues pour optimisation (lazy loading automatique). 

SingleChildScrollView
    Pour un seul widget dont l'affichage risque de dépasser le viewport. Par exemple, lorsque le clavier s'affiche et la taille du viewport est réduite.
    child: (widget), scrollDirection: Axis.vertical/horizontal, 
    Column peut être mis comme child, puis insérer comme ses childrens les éléments d'une liste. 

PageView
    Liste scrollable où chaque enfant occupe l'entiereté du viewport.
    scrollDirection: Axis.vertical/horizontal, 
    Utiliser PageView.builder() pour les listes longues pour optimisation (lazy loading automatique). 

GridView
    Grille de widgets scrollable avec optimisation de performances pour les grandes listes.

double width = MediaQuery.sizeOf(context).width;
double height = MediaQuery.sizeOf(context).height;

//Largeur et hauteur du safe area (zone manipulable) de l'écran
var padding = MediaQuery.paddingOf(context);
double newheight = height - padding.top - padding.bottom;

//Sans contexte
// First get the FlutterView
FlutterView view = WidgetsBinding.instance.platformDispatcher.views.first;
// Dimensions in physical pixels (px)
Size size = view.physicalSize;
double width = size.width;
double height = size.height;
// Dimensions in logical pixels (dp)
Size size = view.physicalSize / view.devicePixelRatio;
double width = size.width;
double height = size.height;

    Scaffold

    // Usage classique d'un scaffold
Scaffold(
  appBar: AppBar(title: const Text('My App')),
  body: const Center(child: Text('Content Area')),
  floatingActionButton: FloatingActionButton(
    onPressed: () {},
    child: const Icon(Icons.add),
  ),
);

    NOTES:
    * Scaffold doit se trouver à l'intérieur d'un MaterialApp, sinon il y a un problème de Directionality.
    * Scaffold est déjà scrollable. Mettre un autre scrollable (ex: SingleChildScrollView) à l'intérieur génère une exception.
     

    PageView

    // Usage classique d'un PageView
import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({Key? key}) : super(key: key);

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        body: PageView(
          children: <Widget>[
            Container(
              color: Colors.orange,
              child: const Center(child: Text('Page 1', style: TextStyle(fontSize: 48))),
            ),
            Container(
              color: Colors.redAccent,
              child: const Center(child: Text('Page 2', style: TextStyle(fontSize: 48))),
            ),
            Container(
              color: Colors.blueGrey,
              child: const Center(child: Text('Page 3', style: TextStyle(fontSize: 48))),
            ),
          ],
        ),
      ),
    );
  }
}
