    Fanomanana

    Alohan'ny hampiasa SQLite ao @ Godot dia mila ampidirina ny plugin Godot-SQLite. 

    1. Sokafana ny praograma Godot
2. Mankao @ menoa AssetLib ery ambony
3. Tadiavina ny hoe "godot-sqlite"
4. Klikena ilay singa Godot-SQLite asehony
5. Manao "Download" (eo @ 60Mo ny habeny)
6. Klikena ny bokotra "Install"
7. Rehefa vita ny installlation dia vata "addons" miampy ao @ tetikasa
8. Mandeha ao @ Project > Projects settings > tab Plugins, dia cocher-na plugin Godot SQLite

    Mba tsy hanokafana/hanakatonana ny bazy isaky ny miova script dia tsara raha atambatra amina singleton anankiray. 

    1. Manamboatra script manokatra ny bazy. Ohatra: res://DB.gd
extends Node
var sql: SQLite
func _ready():
    sql = SQLite.new()
    sql.path = "user://labase.db"
    var ok = sql.open_db()
    if not ok:
        push_error("Échec d'ouverture DB: " + error_string(ok))
    //tokony atao eto ny fanamboarana ny structure bazy sql (DDL)

2. Atao Autoload ny script mba ho lasa singleton
Mankao @ Project > Project Settings > tab Globals > Autoload
Nom: DB
Chemin: res://DB.gd
Cocher "Add"  // azo @ DB ny bazy
