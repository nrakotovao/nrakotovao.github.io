Créer un projet

  1. Ouvrir l'application Solar2D Simulator
  2. Aller dans File > New project...
  3. Indiquer le nom du projet et choisir Blank comme template
  4. Ouvrir le fichier main.lua dans un éditeur de texte. Ce fichier est le fichier principal d'une app Solar2D, et qui est appelé au lancement de l'app.
